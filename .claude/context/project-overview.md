# Project Overview — EWW Connect

## What is EWW Connect?
An internal collaboration platform for an IT company, modeled after Microsoft Teams. It centralizes team chat, channel-based collaboration, file sharing, and meeting scheduling — delivered as a web app and a native desktop app (via Electron).

## Target Users
- IT company employees (all departments)
- Team leads / Managers (channel & member management)
- Admins (organization-wide settings, user provisioning)

## Business Goals
- Reduce dependency on external tools (Slack/MS Teams licensing costs)
- Full control over data (self-hosted)
- Tailored workflows specific to the company's internal processes

## Phases

### Phase 1 (MVP) — Current Focus
- Authentication & Organization setup
- Chat (1-on-1 + group, real-time)
- Channels (team/department-based)
- File sharing
- Meetings/Calendar (scheduling + reminders, no live video yet)

### Phase 2 (Post-MVP)
- Video/Audio calling (WebRTC)
- Screen sharing
- Advanced admin controls (audit logs, compliance)
- Mobile app

## Non-Goals (for now)
- Public/external guest access
- Multi-tenant SaaS (single organization deployment for MVP)
- Third-party app marketplace/bots
